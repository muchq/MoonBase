# Tracy

C++ ray tracer implementation following [Computer Graphics from Scratch](https://gabrielgambetta.com/computer-graphics-from-scratch/)

It's a port of a [port](../tracy_go) of a [port](../scraphics) of a [port](https://gist.github.com/aaylward/5a0aabd1def217b1f1319a4695429adf) of an [implementation](https://github.com/aaylward/tracer_school) described in an [ebook](https://gabrielgambetta.com/computer-graphics-from-scratch/02-basic-raytracing.html)

## Features
 - [x] spheres
 - [ ] cubes
 - [ ] pyramids
 - [x] reflections
 - [x] shadows
 - [x] multiple light sources
 - [x] starry background
 - [ ] import background image
